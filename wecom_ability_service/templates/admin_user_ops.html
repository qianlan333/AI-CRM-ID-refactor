统一用户运营看板 v2

这版看板围绕手机号主线管理 CRM 用户投影池，聚焦手机号、是否已加微、班期和激活状态，统一查看并校准运营数据。

手机号 + 班期导入

独立导入入口。支持粘贴或上传 `.xlsx / utf-8 文本`，格式固定为 `手机号 + 班期`，例如 `13800138030,5期`。同手机号重复导入时覆盖班期，不通过导入直接写入 external_userid。

准备导入。班期保留原始业务表达，是否已加微继续通过现有绑定关系推导。

手机号 + 激活状态导入

独立导入入口。格式固定为 `手机号 + 激活状态`，激活状态只允许 `激活 / 未激活`。同手机号重复导入时按最后一条覆盖，不通过导入直接写入 external_userid。

准备导入。仅接受“激活 / 未激活”，非法值会直接报错。
