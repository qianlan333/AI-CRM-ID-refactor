## Inventory Notes

- Commands that can affect production data remain retained or marked needs manual review.
- No command in this inventory is approved for deletion in D8.5.
- Production database migration is not executed by this planning gate.
- Any future command removal must be handled as a separate implementation and acceptance stage.
