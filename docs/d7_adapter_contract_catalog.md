## Shared Contract Rules

- Every external adapter must return a stable error shape.
- Every adapter must emit an audit event before and after attempted side effects.
- Every production mode requires explicit environment enablement.
- Every staging mode must support fake or dry-run execution.
- Every rollback path is flag-disable first, then route back to legacy fallback.
