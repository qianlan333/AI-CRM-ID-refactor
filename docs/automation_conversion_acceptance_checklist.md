# 自动化转化验收 Checklist

## 问卷验收项

- [ ] 自动化转化问卷支持任意数量关键题
- [ ] 单题支持多命中答案
- [ ] 切换题目后不再默认自动选第一个答案
- [ ] 问卷必须包含必填手机号题
- [ ] 提交成功后手机号正常保存
- [ ] 问卷只负责普通跟进 / 重点跟进首次分流

## 6 池子验收项

- [ ] 新用户池存在且可进入
- [ ] 未激活普通池存在且可进入
- [ ] 未激活重点跟进池存在且可进入
- [ ] 激活普通池存在且可进入
- [ ] 激活重点跟进池存在且可进入
- [ ] 沉默池存在且可进入
- [ ] 同一用户同一时间只能有一个当前池子
- [ ] 每次切池都记录入池时间
- [ ] 每个可沉默池都能配置停留天数

## 侧边栏验收项

- [ ] 能看到当前池子
- [ ] 能看到当前是否激活
- [ ] 能看到当前跟进类型
- [ ] 能看到问卷初判来源
- [ ] 能看到人工覆盖来源
- [ ] 未激活场景下可普通/重点互切
- [ ] 已激活场景下可普通/重点互切
- [ ] 人工确认成交后退出营销

## OpenClaw 群发验收项

- [ ] OpenClaw 可直接调用 CRM 按池子群发
- [ ] 新用户池可群发
- [ ] 未激活普通池可群发
- [ ] 未激活重点跟进池可群发
- [ ] 激活普通池可群发
- [ ] 激活重点跟进池可群发
- [ ] 沉默池不可群发
- [ ] 发送记录正常落库

## 焦点消息 webhook 验收项

- [ ] 重点跟进池用户来消息会触发 OpenClaw webhook
- [ ] 普通池用户来消息不会触发
- [ ] payload 包含客户档案、问卷摘要、标签、最近聊天、当前阶段
- [ ] 失败会留日志，不影响消息入库

## 激活回写验收项

- [ ] 可按手机号回写激活
- [ ] 未激活普通池 -> 激活普通池
- [ ] 未激活重点跟进池 -> 激活重点跟进池
- [ ] 已激活重复回写只刷新最近激活时间
- [ ] 找不到手机号时返回明确错误

## 问卷 webhook 验收项

- [ ] 提交成功后会外发 webhook
- [ ] 外发字段固定为 mobile / userid / unionid
- [ ] webhook 失败不影响问卷提交成功

## 成交退出验收项

- [ ] 人工确认成交后状态变成 `converted/enrolled`
- [ ] 成交后不再 eligible for conversion
- [ ] 已排队的自动化动作被取消或不再继续
