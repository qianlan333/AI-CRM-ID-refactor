# AI-CRM 客户侧边栏 V2 Demo 逻辑说明

## 当前确认范围

本版本只保留侧边栏内高频查看和高频操作的模块：

1. 顶部基准信息
2. 核心画像
3. 问卷
4. 商品
5. 订单
6. 素材
7. 聊天记录

已去掉以下内容：

- AI 自动生成建议
- 建议跟进
- AI 话术推荐
- 当前版本里的客户分层展示
- 独立的大块自动化池信息
- 独立自动化操作区

## 顶部信息

顶部只展示：

- 头像
- 客户昵称
- 手机号
- 手机号已绑定状态
- 外部联系人 ID
- 更换手机号
- 当前所在工作流的小型展示

工作流展示改成小型一行，例如：

```text
暑期阅读转化 SOP · 第 3 天 · 重点跟进节点
```

不再使用大块的“自动化池 / 自动化运营池”标题。

## 核心画像

核心画像作为默认首页。字段后续从问卷关键字段中绑定，当前 Demo 用模拟数据展示：

- 用户来源
- 行业
- 行业描述
- 具体需求点
- 孩子年级
- 预算区间
- 沟通偏好

这里不做客户分层，也不做 AI 自动判断。

## 问卷

问卷模块展示客户提交过的多份问卷。每份问卷包含：

- 问卷名称
- 提交时间
- 完成状态
- 问卷说明
- 题目和答案

操作：

- 查看答案
- 原始记录

## 商品

商品模块展示后台配置好的商品。每个商品包含：

- 商品名称
- 价格
- 课程/商品说明
- 适配依据
- 发送介绍
- 查看详情

适配依据来自问卷字段或业务规则，不使用 AI 自动建议。

## 订单

订单模块展示历史订单。每个订单包含：

- 订单号
- 支付状态
- 金额
- 商品
- 支付时间
- 交付状态
- 售后/退款信息

操作：

- 查看订单详情
- 复制订单号

## 素材

素材模块新增为独立 Tab，连接素材库。素材类型分三类：

1. 图片素材
2. 小程序素材
3. PDF 素材

素材模块支持：

- 分类切换
- 搜索素材名称、用途、标签
- 预览素材
- 发送给客户

图片素材适合课程海报、方法长图、反馈截图、朋友圈图。

小程序素材适合试听预约、测评入口、课程介绍、打卡入口、支付确认。

PDF 素材适合课程大纲、诊断报告、家长手册、付款售后说明。

## 聊天记录

聊天记录默认展示最近聊天，按时间从近到远排列。后续真实开发时可沿用现有聊天记录接口，并支持加载更多。

## GitHub Pages 使用方式

这个 Demo 是单文件 HTML，可以直接放入仓库，例如：

```text
docs/demos/customer-sidebar-v2-material-demo.html
```

如果仓库启用了 GitHub Pages，可通过 Pages URL 直接展示给团队或客户看。

当前环境的 GitHub 连接只提供读取/检索能力，没有写入仓库、创建分支或提交文件的工具。因此本轮先生成可直接提交的静态 HTML 文件和说明文档。
